03 / 24
harness
two axes
▸ feedforward + feedback

Two complementary control flows.

▸ Guide (feedforward)
Shape the model's view before it acts.
AGENTS.md · skills · API conventions · LSP hints · codemod recipes. Computational or inferential.
▸ Sensor (feedback)
Observe results after it acts; produce signals.
clippy · cargo test · review agent · runtime SLOs. Signals carry LLM-friendly correction hints.

Skip one and the agent either repeats mistakes (no feedback) or knows the rules but never learns whether they applied (no feedforward).

#[harness::skill(
    name = "format-rust",
    license = "Apache-2.0",
    allowed_tools = "Bash(cargo:fmt)",
    harness(kind = "computational", risk = "read-only"),

async fn format_rust(ctx: &mut Context, w: &mut World) -> Result<(), SkillError> {
    w.runner.exec("cargo fmt --all").await?;
    Ok(())
}

08 / 24
harness
agent loop
▸ smallest useful agent

An entire agent in 12 lines.

use harness::prelude::*;
use harness_models::{OpenAiCompat, providers};
use harness_tools_fs::{ReadFile, ListDir, WriteFile};

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let model = OpenAiCompat::with_key(providers::DEEPSEEK, "deepseek-v4-pro", std::env::var("DEEPSEEK_API_KEY")?);
    let mut world = harness_context::default_world(".");
    AgentLoop::new(model)
        .with_tool(Arc::new(ReadFile)).with_tool(Arc::new(ListDir)).with_tool(Arc::new(WriteFile))
        .run(Task::from("summarise this repo into NOTES.md"), &mut world)
        .await?;
    Ok(())
}
09 / 24
harness
blueprint
▸ stripe minions pattern

Hybrid state machines:
deterministic + agent nodes.

Blueprint::new(WorktreeSandbox::new(repo))
    .add("bump_deps",   Node::agent(/* read Cargo.toml, propose upgrades */))
    .add("fmt",         Node::deterministic(|w| w.runner.exec("cargo fmt --all")))
    .add("clippy_fix",  Node::deterministic(|w| w.runner.exec("cargo clippy --fix")))
    .add("test",        Node::deterministic(|w| w.runner.exec("cargo nextest run")))
    .add("changelog",   Node::agent(/* write changelog from diff */))
    .edges([("bump_deps", "fmt"), ("fmt", "clippy_fix"), ("clippy_fix", "test")])
    .branch_on_failure("test", retry_cap=2)

The LLM never formats code or runs git ops. Token spend drops; reliability climbs.

compactor
▸ context is finite

Five-stage progressive compaction.

> 60%
BudgetReduce
Drop redundant tool results.
> 70%
Snip
Hash old file reads.
> 80%
Microcompact
Cheap-model summary.
> 90%
Collapse
Merge to file list + excerpts.
> 95%
AutoCompact
Main model rewrite.

Stages are cumulative. PreCompact / PostCompact hooks let you protect "identity" docs.

getting started
▸ zero hand-editing

From clone to live agent in 90 seconds.

$ git clone github.com/liliang-cn/harness
$ cargo install --path crates/harness-cli

$ harness new my-agent --local        # auto-wires [patch.crates-io]
$ cd my-agent
$ export DEEPSEEK_API_KEY=sk-…
$ cargo run

→ Done { text: Some("..."), iters: 2 }

Generated main.rs ships with one #[skill], one #[tool], and a working AgentLoop. Swap providers by changing one line.
